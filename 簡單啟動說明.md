# 簡單啟動方式

## 平常展示只要這樣

1. Unity 開專案。
2. Unity 按 Play。
3. 雙擊 `啟動展示.bat`。
4. Python 裡選 CFG 後按 Start，或用 Replay 播 `.bin` / `.csv`。

Python 的 Unity Output 已經預設打開，會送到：

```text
127.0.0.1:5055
```

## Unity 第一次設定

1. 建立 Unity 3D 專案。
2. 把 `UnityDemoScripts` 裡全部 `.cs` 檔拖進 Unity 的 `Assets/Scripts/`。
3. Unity 上方選單點：

```text
Area Scanner > Create Demo Scene
```

4. 按 Play。

之後就不用再手動建立 `Radar Demo`、`Trail Floor`、Camera、Light。

## 展覽前測試

建議先用 Replay 測試：

1. Unity 按 Play。
2. 雙擊 `啟動展示.bat`。
3. Python GUI 的 Replay 選 `.bin` 或 `.csv`。
4. 按 Start Replay。

如果 Unity 方塊會動，代表 Unity 接收正常。
